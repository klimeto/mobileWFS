Map window

The map window consists full OpenStreetMap, Map tools and Main menu.

OpenStreetMap

Map tools

Basic map tools as zoom in / out are available at the top left corner. In addition the layer switcher offers two types of base map data, OSM Online is using the OpenStreetMap Carto and OSM Offline created by downloading the PNG tiles from the same OSM server into the device storage.

Main menu

Locate

The locate button will determine your position using the GNSS sensor and mobile / WIFI networks if available, center the map on calculated latitude and longitude and add the location marder. Each time you change your location the marker will be moved by pressing Locate button.

Observe

The observe button will add new observation on the location determined by the device sensors.

Insert

The insert button will add new observation on the map position you choose by clicking. The App informs the user when insert mode is active by displaying the green block with the text Manual insert mode active - touch map to insert marker at the bottom of the map window. In order to disable the inserting mode you will have to press the Insert menu item.

Capture

The capture button will trigger the map tile capturing function, which will download the tiles of the OSM map for a particular area defined by the current map window extent and zoom from online OSM server. This map can be then used in the offline mode on the field. The App informs the user about active capturing mode by displaying the red block with the text Capturing the map - it will be available for offline use at the bottom of the map window. In order to disable the inserting mode you will have to press the Capture menu item.

Editor window

The editor window provides editing for to collect attributes for observations.

Observation attributes

Photograph

The photograph block is placed at the top left section of the editor window, initially displaying the Proplant logo thumbnail as the plant photograph. In order to display the original image you have to click on the thunbnail what will open a full screen image window with the button launching the camera available only for the owner.

Plant name

Palnt name field displays initially text Plant name not defined and after the click opens the plant name list full screen window where you can either scroll or search for a latin name of the protected plant. The list of protected species is based on the Annex 5 of Edict 24/2003.

Site configuration

Site configuration describes whether the observation represents a single of a group of plants.

Observation date

Observation date represents the time when the observation was measured. The default value is NOW

Site description

Site description offers a free text option to provide any additional information that might be important for a particular observation from the user point of view.

Certainity

Certainity field provides information about the data quality assessed by the user who collected the observation by selecting one of the value: high, medium, low, unknown.
